=== Gabfire Social Mashup Plugin ===



== Description ==

Displays a mashup of specified social platforms. 

= Reference =



== Installation ==
Best is to install directly from WordPress. If manual installation is required, please make sure to put all of the plugin files in a folder named `gabfire-socialmashup-plugin` (not two nested folders) in the plugin directory, then activate the plugin through the `Plugins` menu in WordPress.
